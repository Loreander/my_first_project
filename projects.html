Markdown Preview

The Markdown Preview web app simply converts a user string into markdown text. Markdown text is a shorthand way of defining certain html contents. If the user types in '# testing', it will display 'testing' using the h1 stylings. There have been many iterations of this website. The most recent update took away the 'render' button. In its place, the text will automatically render on keyup.

Pick a Number

The Pick a Number web app will ask you to guess a number from 1-100. Then you will tell the computer if your number is higher or lower. Eventually the computer will guess your number!

Guess my Number

Its the opposite of 'Pick a Number'. In this web app, you will try to guess the computer's number between 1-100. How quickly can you guess the computers number?!

Pomodoro Timers

We were tasked with creating a timer that could be used to monitor time on task and set reasonable time goals. This timer can be set using the seconds, minutes, hours, up to 12 hours, 59 minutes, and 59 seconds.

Yelpington

The Yelpington app is designed to use AJAX and JSON files to populate a website with restaurant information and location. The app uses "Open Street Maps API" to display a map of the area surrounding the restaurent in question as well as a marker over its exact location. Given that a server is required there is currently no access to this app.
